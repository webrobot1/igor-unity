## AI-Game-Developer-Config.json

Конфиг Unity-плагина `com.ivanmurzak.unity.mcp` (MCP-сервер внутри редактора, через который Claude/AI делает `assets-*`, `gameobject-*`, `script-execute` и т.д.). Плагин читает файл по фиксированному пути `Resources/AI-Game-Developer-Config.json` — не удалять и не перемещать.

Содержит:
- `host` — TCP endpoint MCP-сервера внутри Unity (по умолчанию `http://localhost:52475`).
- `enabledTools` / `enabledPrompts` / `enabledResources` — allowlist того, что плагин экспонирует наружу. Хочешь запретить AI определённую операцию (например `script-delete`) — убери её имя отсюда; плагин просто не зарегистрирует эту команду и MCP-клиент даже не увидит её в списке.
- `timeoutMs`, `keepConnected`, `logLevel` — технические настройки соединения.

Конфиг подключения **со стороны Claude** (адрес сервера, авторизация, permissions) живёт отдельно в `~/.claude/settings.json` — этот файл и наш Unity-конфиг работают в паре, но не дублируют друг друга.
